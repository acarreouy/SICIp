Introducción


A través de esta herramienta puede encontrar rápidamente investigaciones relevantes y confiables, identificar expertos y acceder a datos y métricas confiables de nuestra revista para respaldar su estrategia de investigación. Tratamos de combinar datos artículos, autores y citas, con datos enriquecidos y contenido vinculado.


Esta página trabaja sobre lenguajes:

HTML, para la visualización de páginas estáticas como esta.
PHP, para realizar consultas a la base de datos y visualización de páginas dinámicas.
CSS, para el manejo de los estilos en las páginas y CSS-GRID para la maquetación de tablas.
MYSQL, para el desarrollo de las bases de datos.
HIGHCHARTS, para la visualización de gráficas.
